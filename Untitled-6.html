Elementos Visuales

Elementos Visuales parte 1



Entrada tipo TEXTO sirve para agregar caracteres alfanumericos, como nombres, apellidos, usuarios, cuidades, etc. Son muy importantes en los formularios



Entrada tipo EMAIL sirve para agregar datos que haga referencia a correos electronicos.



Entrada tipo PASSWORD es para utilizar texto pero que al ingresar informacion sea de manera oculta, en pocas palabras aparecera con asteriscos o circulos haciendo privada la informacion.



Entrada tipo NUMBER es para agregar datos numericos y dependiendo al navegador, aparecera un elemento para aumentar o disminuir el numero mas facilmente.



Entrada tipo SEARCH que se utiliza para agregar a la paginas donde se tiene un filtro como su nombre lo indica, una busqueda, y a diferencia del

Entrada tipo TEL es para ingresar datos que sean numericos (telefonico).



Entrada tipo URL es para agregar ligas, URL o vinculos.



Entrada tipo RANGE es para generar una barra para elegir un dato, normalmente se utiliza un forma online en las tiendas para seleccionar un valor
